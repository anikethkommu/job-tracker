# ⚡ JobTrackr — Smart Job Application Tracker

A clean, beginner-friendly web app to track all your job applications in one place.  
Built with **HTML, CSS, and Vanilla JavaScript** — no frameworks, no installations needed.

---

## 🖼️ Preview

> Open `index.html` in your browser and you're good to go!

---

## ✨ Features

- ➕ **Add applications** with company, role, date, status, and notes
- 🎨 **Status badges** — Applied, Interview, Offer, Rejected (colour coded)
- 📊 **Live dashboard** showing counts per status
- 🔍 **Search** by company or role name
- 🗂️ **Filter** by status and **sort** by date or company name
- ✏️ **Edit** applications via a smooth modal popup
- 🗑️ **Delete** with confirmation prompt
- 💾 **Data saved** in browser `localStorage` — no backend needed
- 📱 **Responsive** — works on mobile and desktop
- 🔔 **Toast notifications** for add / edit / delete actions

---

## 📁 Project Structure

```
job-tracker/
│
├── index.html      → App layout and structure
├── style.css       → All styling (CSS variables, cards, modal, responsive)
└── script.js       → All logic (add, edit, delete, search, filter, stats)
```

---

## 🚀 How to Run Locally

1. **Download or clone** this repository
2. Open the project folder
3. Double-click `index.html` — it opens directly in your browser
4. No server, no npm, no setup needed ✅

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and form elements |
| CSS3 | Styling, layout (Grid/Flexbox), animations |
| JavaScript (ES6+) | Logic, DOM manipulation, localStorage |
| Google Fonts | Syne (headings) + DM Sans (body) |
| localStorage API | Persist data without a backend |
